Rust 的可见性

学习链接

私有: pub(in self) , 以当前模块为根的子树可以访问
公有: pub , 无限制，哪里都可以访问

简写形式:
pub(crate) 相当于pub(in crate)
pub(super) 相当于pub(in super)
pub(self) 相当于pub(in self)

Rust 的路径形态
x;
x::y::z;

crate 当前盒
super 父级模块
self 当前模块
$crate 当前宏所在的盒

impl, trait 关键字的作用

学习链接

impl：定义结构体成员方法

  1. impl 关键字在strut， enum或者trait对象上实现方法调用语法
  2. 关联函数（associated function）的第一个参数通常为self参数

self参数有几种变体:

  1. self, 允许实现者移动和修改对象，对应的闭包特性为FnOnce
  2. &self, 既不允许实现者移动对象也不允许修改，对应的闭包特性为Fn
  3. &mut self，允许实现者修改对象但不允许移动，对应的闭包特性为FnMut
  4. 不含self参数的关联函数称为静态方法(static method)

trait：定义接口
代码来源

struct S;

impl S {
    fn f() { println!("S"); }
}

trait T1 {
    fn f() { println!("T1 f"); }
}

impl T1 for S {}

trait T2 {
    fn f() { println!("T2 f"); }
}

impl T2 for S {}

fn main() {
    S::f();
    <S as T1>::f();
    <S as T2>::f();
}

/*
output:
S
T1 f
T2 f
/*

}
